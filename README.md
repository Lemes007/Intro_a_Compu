# Introdução à Computação

## Sobre o Repositório

Este repositório foi criado com o propósito de centralizar e organizar todos os projetos desenvolvidos durante a disciplina de Introdução à Computação, ministrada pela Professora Kadidja no curso de Ciência da Computação do Centro Universitário de Brasília (CEUB), Campus Taguatinga.

## Objetivo

O repositório serve como um portfólio acadêmico dos conhecimentos adquiridos ao longo da disciplina, documentando o progresso educacional e as competências desenvolvidas em fundamentos de computação.

## Conteúdo

Este espaço contém:

- Projetos práticos desenvolvidos durante as aulas
- Exercícios de programação e conceitos fundamentais
- Documentação dos trabalhos realizados
- Exemplos de código relacionados aos tópicos abordados

## Instituição

- Universidade: Centro Universitário de Brasília (CEUB)
- Campus: Taguatinga
- Curso: Engenharia de Software
- Disciplina: Introdução à Computação
- Professora: Kadidja
