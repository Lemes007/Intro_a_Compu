# 💻 Introdução à Computação | Engenharia de Software

Este repositório foi desenvolvido como **portfólio acadêmico** da disciplina de **Introdução à Computação** do **UniCEUB**, reunindo todas as atividades práticas, pesquisas, relatórios e estudos realizados ao longo do semestre.

O objetivo é documentar o processo de aprendizagem, evidenciar a aplicação dos conhecimentos adquiridos e demonstrar o desenvolvimento de competências técnicas relacionadas à computação, ao uso de tecnologias digitais e às boas práticas de documentação e versionamento de projetos.

# 📚 Sobre a Disciplina

A disciplina de **Introdução à Computação** fornece os fundamentos necessários para a formação em Engenharia de Software, abordando conceitos essenciais sobre computadores, sistemas operacionais, redes de computadores, internet, segurança da informação, infraestrutura tecnológica, transformação digital e tendências da tecnologia da informação.

Além dos aspectos técnicos, a disciplina incentiva o desenvolvimento de competências relacionadas à organização, pesquisa, resolução de problemas e utilização de ferramentas colaborativas, preparando o estudante para os desafios acadêmicos e profissionais.
