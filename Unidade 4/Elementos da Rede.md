# Comunicação de Dados

## Objetivo

Compreender os principais elementos envolvidos na comunicação de dados, identificando a função do emissor, receptor, meio de transmissão e protocolos utilizados para que as informações sejam enviadas e recebidas corretamente.

---

## Emissor

O **emissor** é o dispositivo responsável por gerar e enviar uma mensagem para outro dispositivo através de uma rede de comunicação.

### Exemplos

- Computador
- Notebook
- Smartphone
- Câmera de segurança
- Sensor

---

## Receptor

O **receptor** é o dispositivo que recebe a mensagem enviada pelo emissor e realiza a interpretação dos dados recebidos.

### Exemplos

- Computador
- Servidor
- Smart TV
- Impressora
- Smartphone

---

## Meio de Transmissão

O **meio de transmissão** é o caminho por onde os dados trafegam entre o emissor e o receptor. Ele pode ser físico ou sem fio.

### Exemplos

- Cabo Ethernet
- Fibra óptica
- Wi-Fi
- Bluetooth

---

## Protocolos

Os **protocolos** são conjuntos de regras que garantem que a comunicação entre os dispositivos aconteça corretamente, definindo como os dados serão enviados, recebidos e interpretados.

### Exemplos

- TCP/IP
- HTTP
- HTTPS
- FTP
- SMTP

---

# Resumo

| Elemento | Função | Exemplos |
|----------|---------|----------|
| **Emissor** | Envia a informação | Notebook, celular, câmera |
| **Receptor** | Recebe a informação | Computador, servidor, Smart TV |
| **Meio de Transmissão** | Transporta os dados | Cabo Ethernet, fibra óptica, Wi-Fi |
| **Protocolo** | Define as regras da comunicação | TCP/IP, HTTP, HTTPS, FTP |

---

## Conclusão

A comunicação de dados depende da atuação conjunta do emissor, do receptor, do meio de transmissão e dos protocolos de comunicação. Cada elemento possui uma função específica para garantir que as informações sejam transmitidas de forma correta, segura e eficiente entre diferentes dispositivos.
